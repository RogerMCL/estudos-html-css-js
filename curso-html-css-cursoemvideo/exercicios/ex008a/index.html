Tipo de formatações

Negrito/Destaque

Nesta frase, temos um termo em negrito usando a tag B (não-semântica).

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica).

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não-semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto grande e pequeno

Podemos fazer textos grandes com a tag obsoleta BIG (já não é possível usar) e textos pequenos com a tag SMALL

Texto marcado

Podemos criar textos marcados com a tag MARK.

Posso mudar a cor amarela do MARK através do comando STYLE no head do programa.

Texto deletado

Podemos marcar um texto como excluído, como forma dele ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois

Existe uma forma não-semântica de sublinhar um texto com a tag U.

Texto sobrescito

Exemplo: x12 + 3

Texto subscrito

Exemplo: H2O
